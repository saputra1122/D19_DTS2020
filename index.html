Pelatihan HTML dan CSS akan mempelajari teori dan praktik pemrograman front-end secara daring/online (tampilan halaman web), peserta akan mempelajari cara membuat suatu halaman web modern dan beberapa animasi seperti formulir, tombol 3D, desain responsif menyesuaikan dengan perangkat pengguna.
